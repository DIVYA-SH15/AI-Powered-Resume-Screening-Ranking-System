# AI-Powered-Resume-Screening-Ranking-System

## Project Overview

This project automates the candidate screening process by comparing resumes against a job description using Natural Language Processing (NLP).

The system calculates similarity scores and ranks candidates based on how closely their skills match the job requirements.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Cosine Similarity

## Workflow

1. Load resumes
2. Load job description
3. Convert text into numerical vectors using TF-IDF
4. Calculate similarity scores
5. Rank candidates
6. Recommend top candidates

## Features

* Resume screening automation
* Candidate ranking
* Job-description matching
* Explainable scoring system

## Business Benefits

* Reduces manual resume screening time
* Improves recruitment efficiency
* Helps HR teams identify suitable candidates quickly
* Provides transparent candidate ranking

## Conclusion

The system demonstrates how NLP can be applied to recruitment processes by automatically ranking candidates based on job relevance.
